## 📖 About

**MultiTool** adalah aplikasi CLI (Command Line Interface) serbaguna berbasis Python yang dirancang untuk mempermudah pengelolaan komputer Windows. Dengan **42+ fitur** yang dikemas dalam satu program, MultiTool menyediakan antarmuka terminal yang profesional dengan animasigradient warna, dan dukungan **18 bahasa**.

> 💡 Cocok untuk pengguna yang ingin mengella sistem, jaringan, keamanan, dan optimasi komputer tanpa harus membuka banyak aplikasi berbeda.

## 🚀 Quick Start

### Prerequisites

- **Pytho3.7+** — [Download Python](https://www.python.org/downloads/)
  > ⚠️ **PENTING:** Centang **"Add Python to PATH"** saat instalasi!
- **Windows 10/11** (Beberapa fitur khusus Windows)

### Instalasi

1. **Clone repository:**
   ```bash
   git clone https://github.com/SOBING4413/MultiTools.git
   cd MultiTools
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Jalankan program:**
   ```bash
   python multitool.py
   ```

### Cara Cepat (Windows)

Cukup double-click salah satu file berikut:

| File | Deskripsi |
|------|-----------|
| `RUN.bat` | Jalankan MultiTool (mode normal) |
| `RUN_ADMIN.bat` | Jalankan sebagai Administrator (akses penuh) |

> 💡 **Tip:** Jalankan sebagai **Administrator** untuk mengakses semua fitur (WiFi password, firewall, dll).

## ⚠️ Important Notes

- **Beberapa fitur memerlukan akses Administrator** (WiFi password, firewall, port scanning, dll). Gunakan `RUN_ADMIN.bat` atau jalankan terminal sebagai Admin.
- **Fitur khusus Windows:** Sebagian besar fitur dirancang untuk Windows 10/11. Beberapa fitur dasar (system info, network) juga bisa berjalan di Linux/macOS.
- **Speed Test:** Menggunakan metode download file untuk mengukur kecepatan. Hasilnya mungkin berbeda dengan speedtest.net.
- **Privacy Cleaner:** Berhati-hati saat menggunakan fitur ini karena akan menghapus file secara permanen.
